# Web Technologies Lab Tasks | Abdul Rafay - BSIT 🚀

> A structured collection of web development lab tasks focusing on foundational HTML5 and CSS3 implementations.

This repository contains a series of web programming exercises developed for the March 26th lab session. It features 12 distinct programs that explore various aspects of front-end structure and design, all consolidated into a single accessible directory.

## 📂 Project Structure

The project is organized into modular HTML files and utilizes a centralized stylesheet, making it straightforward to navigate through the individual lab requirements:

| File / Directory | Description |
| :--- | :--- |
| `index.html` | The main landing page that serves as a directory, linking to all 12 lab tasks. |
| `program1.html` – `program12.html`| Individual HTML documents containing the specific code implementations for tasks 1 through 12. |
| `style.css` | The shared stylesheet responsible for the visual design, typography, and layout of all the HTML pages. |
| `.github/workflows/` | Configurations for continuous integration and automated GitHub Pages deployment. |

## 🎯 Learning Objectives

This lab assignment focuses on building practical competencies in front-end development, specifically:
* Writing clean, semantic, and well-structured HTML documents.
* Applying custom styling, colors, and layout configurations using pure CSS.
* Managing multiple interconnected web pages within a unified project architecture.
* Utilizing automated CI/CD workflows for deploying static web projects.
